## О проекте
Сервис автоматизирует обработку заявок с рекламных форм ВКонтакте. Вместо ручного переноса информации из лид-ыормы в CRM, скрипт самостоятельно забирает данные из почты, создаёт сущности в CRM и уведомляет команду.

**Сценарий работы:**
1. Клиент оставляет заявку в **VK Forms**.
2. Письмо попадает на выделенный **IMAP-ящик**.
3. Скрипт парсит тело письма, извлекает контакты и комментарии.
4. В **Bitrix24** создаётся Контакт и Сделка с заполненными полями.
5. Менеджер получает сообщение в **Telegram** с кнопками для быстрого перехода.

## Стек технологий
- **Язык:** Python 3.9+
- **Интеграции:** Bitrix24 REST API, Telegram Bot API, IMAP4
- **Архитектура:** Модульная (Client-Parser-Worker)
- **Хранение:** Локальный кэш (SQLite/Text), переменные окружения (.env)
